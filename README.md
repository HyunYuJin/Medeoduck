### Installing / 설치

이 블로그는 Webpack과 Babel 라이브러리를 이용하기 위해 node.js를 설치해야 합니다.

```
npm init
npm install -g webpack webpack-cli
npm install --save-dev @babel/core @babel/preset-env babel-loader
```

## Webpack 실행

```
npx webpack
```
